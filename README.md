# Snake-3D
Игра змейка, реализованная с помощью C++ и OpenGL (GLUT).
## Описание
Игровое поле, отрисовывается средствами OpenGL в отдельном окне, инструкции по игре и статистика игровой сессии отображаются в консоль.
